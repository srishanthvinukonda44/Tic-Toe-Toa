🧩 PulseMatch - Memory Matching Game
PulseMatch is a modern Android memory matching game designed for two players. Players take turns flipping cards to find matching pairs, earn points, and compete for the highest score. The application features a clean UI, real-time score tracking, and turn-based gameplay.

🎮 How to Play
Enter your name and sign in.
Start the game with two players.
Players take turns flipping two cards.
If both cards match, the player earns points.
If cards do not match, the turn switches to the other player.
The player with the highest score wins.
